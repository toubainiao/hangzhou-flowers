## 📝 如何修改网站内容

### 1. 修改赏花地点数据

**文件位置：** `index.html` 中 `flowerSpots` 数组（约第620行开始）

**数据格式：**
```javascript
{
    id: 1,
    name: "太子湾公园",           // 地点名称
    flower: "郁金香",             // 花卉种类
    lat: 30.228,                  // 纬度
    lng: 120.136,                 // 经度
    image: "图片URL",             // 封面图片
    bloomStart: "03-01",          // 开花开始日期
    bloomEnd: "04-15",            // 开花结束日期
    fullBloomStart: "03-15",      // 盛花期开始
    fullBloomEnd: "04-05",        // 盛花期结束
    tips: "游玩建议...",           // 提示信息
    description: "地点描述...",    // 详细描述
    address: "详细地址"            // 导航地址
}
```

**添加新地点步骤：**
1. 打开 `index.html`
2. 找到 `flowerSpots` 数组
3. 在最后一个地点后面添加新数据（注意id要递增）
4. 保存文件

### 2. 修改花期时间

找到对应地点，修改以下字段：
```javascript
bloomStart: "03-01",      // 改为实际开花日期
bloomEnd: "04-15",        // 改为实际结束日期
fullBloomStart: "03-15",  // 改为实际盛花期开始
fullBloomEnd: "04-05",    // 改为实际盛花期结束
```

**日期格式：** `MM-DD`（月-日，补零）

### 3. 修改页面样式

**颜色主题：** 找到 `<style>` 部分（约第10-450行）

主要颜色变量：
```css
/* 渐变色背景 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 主要按钮颜色 */
background: #667eea;

/* 花期状态颜色 */
盛花期: #ff4757 (红色)
初开期: #ffa502 (橙色)
将谢期: #747d8c (灰色)
未开放: #74b9ff (蓝色)
```

### 4. 修改页面标题

找到 `<title>` 标签（第6行）：
```html
<title>杭州赏花日历 - 2025花季指南</title>
```

## 🔧 常见问题排查

### 问题1：地图不显示（空白）

**原因：**
- 高德Key白名单未配置
- Key已过期
- 网络问题

**解决：**
1. 登录 https://console.amap.com/dev/key/app
2. 找到Web端Key（`6b677be2aab2719b595ea12cafbffccc`）
3. 点击「设置」→ 安全域名
4. 添加：`toubainiao.github.io` 和 `*.github.io`
5. 保存，等待5分钟后刷新网站

### 问题2：搜索功能失败

**原因：**
- Web服务Key过期
- 网络问题

**解决：**
1. 登录高德控制台
2. 检查Web服务Key（`66e4c25ae0a3da4b707e417fb0d88b60`）状态
3. 如过期，创建新的Web服务Key
4. 替换代码中的 `SEARCH_KEY` 值

### 问题3：部署失败（push失败）

**原因：**
- 远程仓库有更新未拉取
- 网络问题

**解决：**
```bash
git pull origin main --allow-unrelated-histories
git push origin main
```

### 问题4：网站样式错乱

**原因：**
- CSS语法错误
- 浏览器缓存

**解决：**
1. 检查修改的CSS是否有语法错误
2. 按 `Ctrl+F5` 强制刷新浏览器
3. 或清除浏览器缓存
